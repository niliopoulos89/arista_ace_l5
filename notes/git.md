
# GIT CLI Cheat Sheet

## Initiate New Repo

```bash
git init
```

## Change Branch From Master to Main

```bash
git branch -M main
```

## Add a file to the GIT Repo

```bash
git add filename.txt
```

## Check the GIT Repo Files

```bash
git ls-files
```

## Commit  the GIT Repo Files

```bash
git commit -m "Adding Placeholder File"
```

## Check Current GIT Status

```bash
git status
```
